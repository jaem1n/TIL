- for x in range(1, n+1) : 1부터 n까지 for문 돌리기, 시작≤x<끝이기 때문에 n+1로 설정해야 n까지 해당하는 값을 얻을 수 있다.

        factorial = 1
        for x in range(1, n+1):
        	factorial *= x
